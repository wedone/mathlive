# 数学虚拟键盘

在输入数学公式时需要大量特殊符号。对于有物理键盘的设备，可以使用[快捷键和内联快捷方式](https://mathlive.io/mathfield/guides/shortcuts/)。但在仅有触摸屏且没有物理键盘的设备上，需要另一种方式。

数学虚拟键盘是一个屏幕键盘，用户只需轻触即可输入数学专用符号。

![虚拟键盘预览](https://mathlive.io/assets/images/virtual-keyboard-59910bc35a213891d4d6a5dc14f230e4.webp)

---

## 布局选择器

虚拟键盘包含多个布局，可通过布局选择器切换。默认布局包含最常用的数学符号：`numeric`（数字）、`symbols`（符号）、`alphabetic`（字母）和 `greek`（希腊字母）。

这些布局可以根据需求自定义。了解详情请参见自定义虚拟键盘指南。

---

## 撤销 / 重做 / 粘贴（编辑工具栏）

编辑工具栏显示常用编辑功能的图标：撤销、重做、粘贴、剪切与复制。

如果连接了物理键盘，这些命令也可以通过对应的快捷键调用：`Ctrl/Command + Z`、`Ctrl/Command + Y`、`Ctrl/Command + V`、`Ctrl/Command + X` 与 `Ctrl/Command + C`。

---

## Shift 键

按下 Shift 键（⇧）可以访问按键的替代符号，键盘会在按下时更新以显示这些符号。若有物理键盘，物理键盘上的 `SHIFT` 或 `CAPSLOCK` 也可生效。

部分按键的右上角会显示 shift 后的符号，有些按键虽然没有右上角标记，但仍存在 shift 变体。

---

## 按键变体

按键变体是可以通过长按按键访问的替代符号，通常与主符号相关。要显示某个按键的变体，请长按该按键。

多数按键都有变体，但也有少数没有。

![变体面板示例](https://mathlive.io/assets/images/variant-panel-d5da2fd515fdef5aa13fe0338e2b3d04.webp)
